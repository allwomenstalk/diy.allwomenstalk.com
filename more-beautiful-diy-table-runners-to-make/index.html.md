# 11 More Beautiful DIY Table Runners to Make ...

Continuing our [table decor](https://diy.allwomenstalk.com/coffee-table-decor-ideas-that-dont-require-a-home-stylist/) craft ideas, I give you here our 2nd set of DIY Table Runners to make. These **handmade table runners** will be perfect for making your dining area look pretty for the coming holidays. While some of these are holiday or occasion-specific, there are also a few **DIY table runners** here that you can make for any time of the year. Enjoy!

## 1. Hanky

![Hanky](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/12_300x210.jpg)

This pretty table runner is made of old handkerchiefs. Adorable, right? The best material for this type of DIY project are handkerchiefs that you can find at flea markets or thrift shops; they have beautiful vintage patterns.

[Tutorial](https://www.countryliving.com/diy-crafts/how-to/g1082/buy-or-diy-projects/)

## 2. Bias Tape

![Bias Tape](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/22_485x323.jpg)

Unlike the sewing project from the previous DIY, this particular **handmade table runner** will take a lot longer to make. It's not very hard to make, though, so have no fear. As long as you have pretty fabric and equally pretty double fold bias tape, your table runner will turn out fab.

[Tutorial](https://www.prettyprudent.com/bias-tape-table-runner-2/)

## 3. Rainbow Felt

![Rainbow Felt](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/32-560x403_560x403.jpg)

How fun! This** DIY table runner** is sure to be perfect for a kiddie party. Aside from the colorful felt, you will need Heat & Bond adhesive sheets, sewing machine, scissors, chalk marker, and ruler to make this fun table decor.

[Tutorial](https://goo.gl/1WojSe)

## 4. Quilted Carrot

![Quilted Carrot](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/42_550x393.jpg)

This particular table runner is perfect for an Easter celebration. Of course, if you and your family love carrots, there is no one stopping you from using this table runner all year.

[Tutorial](https://www.bhg.com/holidays/easter/crafts/pretty-easter-crafts-for-adults/)

## 5. Painted Stripes

![Painted Stripes](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/52_550x338.jpg)

Here is an easy way to make a pretty table runner. Get a pre-made canvas table runner, tape painter's tape, and paint stripes. Let dry and you're done.

[Tutorial](https://weseerainbows.wordpress.com/2012/06/05/we-create-striped-table-runner/)

## 6. Pinwheel

_ Yes, that is a table runner. While this type of table runner will not give you a lot of space for food, it will make your table extra photogenic.

Tutorial_

## 7. Scarf

![Scarf](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/72-560x425_560x425.jpg)

This idea is pretty much similar to the handkerchief **DIY table runner** I've mentioned here. However, for this particular craft project, you will be using scarves. Fun, right?

[Tutorial](http://www.madiganmade.com/2012/05/patriotic-no-sew-table-runner-from.html)

## 8. Crop Circles

![Crop Circles](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/82_500x375.jpg)

I find this **handmade table runner** terribly cute. If you have lots of pretty scrap fabric, this is one of the best uses for them. Cut in circles and use them to embellish a pre-made table runner. Easy!

[Tutorial](http://poppyprintcreates.blogspot.com/2009/10/tutorial-crop-circle-table-runner.html)

## 9. Chalkboard

![Chalkboard](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/9a-560x388_560x388.jpg)

A table runner where your guests can doodle or write messages? Sounds fun! To make this, you will simply paint oilcloth with chalkboard paint. Let dry and it's ready for use.

[Tutorial](http://blog.heylook.fi/2012/04/diy-chalkboard-table-setting.html)

## 10. Quilted

![Quilted](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/102_451x321.jpg)

How pretty! If you love to quilt and you're looking for a small project to do, this **DIY table runner** is for you. If you're looking to sew a version perfect for the upcoming holidays, you can let the fabrics do the talking, so to speak.

[Tutorial](http://modifytradition.blogspot.com/2010/03/table-runner.html)

## 11. Photos

![Photos](https://resize.allw.mn/1028x0/filters:format(webp)/filters:quality(70)/content/diy/2012/11/14-560x419_560x419.jpg)

How creative! You simply print old-timey photos, glue to a fabric table runner, and place on top of a table. The result: a great method for showcasing your family's history or those cute photos of you as a kid.

[Tutorial](https://ruffledblog.com/diy-photo-table-runner/)
